# Fundamentos Java

Repositório de estudos para praticar fundamentos de Java, lógica de programação e primeiros conceitos de programação orientada a objetos.

## Visão geral

Este projeto registra minha evolução com Java por meio de exemplos simples, exercícios e organização gradual dos principais conceitos da linguagem.

## Status do projeto

Em evolução.

## Primeiro código

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Olá, mundo!");
    }
}
```

## Exemplos básicos

Variáveis:

```java
String nome = "João";
int idade = 20;
```

Condicional:

```java
if (idade >= 18) {
    System.out.println("Maior idade");
}
```

Repetição:

```java
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
```

## Como executar

Compile o arquivo:

```bash
javac Main.java
```

Execute o programa:

```bash
java Main
```
